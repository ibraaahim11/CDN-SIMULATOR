# 🌐 CDN Simulator

Interactive HTML-based simulation to visualize how Content Delivery Networks (CDNs) work in modern systems.

## 📘 Course Information

- **Topic**: CDNs in Modern Networking
- **Course**: CMPS405: Computer Networks-1
- **Presented To**: Dr. Salma A. Abd elMotaleb

## 🎯 Overview

This simulator demonstrates core CDN concepts, including:

- Anycast-style routing
- Edge caching (cache hit/miss with TTL)
- Load distribution across edges
- Origin server interaction and failure handling

## 🎮 Features

- Adjustable request rate and cache TTL
- Origin server failure simulation
- Live metrics: cache hit rate, latency, origin load
- Real-time request flow visualization

## 📌 Note

Built for educational purposes to illustrate CDN behavior in a simplified, interactive way.
